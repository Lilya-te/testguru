Test Guru

Приложение для прохождения тестов

О проекте

Test Guru - это веб-приложение, созданное на Ruby on Rails, которое позволяет пользователям проходить различные тесты. Проект разработан как учебный в онлайн-школе Thinknetica.

Ключевые особенности

Разнообразие тестов

Большой выбор тестов по различным темам и направлениям.

Простая регистрация

Быстрый процесс регистрации и входа в систему.

Статистика результатов

Отслеживание прогресса и анализ результатов.

Технологии

Приложение разработано с использованием:

  • Ruby on Rails
  • HTML5
  • CSS3
  • JavaScript
